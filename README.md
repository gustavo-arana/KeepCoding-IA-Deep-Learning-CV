#KeepCoding-IA-Deep-Learning-CV
Repositorio de entrega de la práctica de Deep Learning y Computer Vision.

Este repositorio contiene la creación de modelos para resolver el problema de clasificación de enfermedades de la piel utilizando el dataset HAM10000 (Human Against Machine with 10000 training images).

## Entrega Proyecto DL IA V | 1. Modelos Tabular e Imágenes
Contiene la carga y preparación de los datos para el entrenamiento. Se crean, de forma independiente, un modelo tabular y un modelo para el procesamiento de imágenes.

## Entrega Proyecto DL IA V | 2. Modelos Early y Late Fusion
Se implementan las técnicas de Early Fusion y Late Fusion mediante la creación de un modelo multimodal (combinando datos tabulares e imágenes).

## Entrega Proyecto DL IA V | 3. Fine-Tuning
Implementación de la técnica de Fine-Tuning utilizando los modelos VGG16, VGG19 y ResNet para la clasificación de las imágenes del dataset.

## Entrega Proyecto DL IA V | 4. Fine-Tuning + Early Fusion
Desarrollo de un modelo multimodal que combina la técnica de Early Fusion con el Fine-Tuning del modelo ResNet para la clasificación de las imágenes del dataset.
